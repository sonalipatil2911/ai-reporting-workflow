# AI Reporting Workflow: From Raw Sales Data to Executive Business Review

## 📌 Project Overview
This project demonstrates an automated workflow designed to ingest raw operational data, validate calculations deterministically via a spreadsheet engine, extract semantic business risks/trends using LLMs, and visualize performance layers for executive stakeholders.

### ⚙️ The Architecture
1. **Data Layer:** 150 rows of mock sales data injected with hidden trend operational anomalies (e.g., high-spend/low-intent lead generation channels).
2. **Deterministic KPI Engine:** Google Sheets calculation layer tracking conversion rates, CAC, CPL, and RPL against hardcoded business benchmarks.
3. **Semantic Layer (AI Studio):** Google AI Studio implementation leveraging structural JSON output controls to enforce predictable business insights extraction.
4. **Visual Layer:** Interactive cross-filtered Looker Studio Executive Dashboard.
5. **Presentation Layer:** 5-slide modular business review deck built using clean web technologies.

## ⏱️ Efficiency & Performance Impact
* **Manual Processing Baseline:** 2 - 3 Hours per reporting cycle.
* **AI-Assisted Workflow Time:** 30 - 45 Minutes.
* **Operational Optimization:** ~75% reduction in reporting overhead.

## 🛡️ Responsible AI Guardrails
> **Design Note:** This architecture separates deterministic mathematical operations (handled by spreadsheet formulas) from semantic pattern analysis (handled by LLMs). The AI is bounded strictly via a rigid system prompt schema to eliminate mathematical hallucination risks.
